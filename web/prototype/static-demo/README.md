# Hotel RFID Operations Platform MVP

This is the Website-only MVP phase for the Hotel RFID Operations Platform.

Open `index.html` directly in a browser. No framework, dependency install, database, API, or hardware integration is required for this phase.

## Source Of Truth

- Visual direction: `design-reference/DESIGN.md`
- Functional UI requirements: `docs/UI_IMPLEMENTATION_NOTES.md`
- Curated Stitch references: `design-reference/final-desktop/`

## Included Screens

- Dashboard
- RFID Scan
- Linen Master
- Laundry Batches
- Reconciliation
- Device Activity
- Transaction History
- Asset Management

## Phase Boundary

This MVP does not implement Chainway device integration, unified RFID API, event hardening, database persistence, or real fixed reader support.
