# Porta Nusa Hotel RFID Linen Visibility Platform — Shareable Documentation

This folder contains partner-safe documents for the Porta Nusa Hotel RFID Linen Visibility Platform. These files may be shared with hotel management, potential partners, or project stakeholders.

## Documents

### porta-nusa-rfid-documentation-package.pdf

**Purpose:** Complete technical and operational documentation for the platform.
**Audience:** Hotel IT staff, project sponsors, technical evaluators.
**Contents:** System overview, architecture, RFID workflow descriptions (STOCK_COUNT, SEND_TO_LAUNDRY, RETURN_FROM_LAUNDRY), Simulation Mode guide, Hardware Mode guide, and demo operator checklist.

### porta-nusa-solution-faq.pdf

**Purpose:** Answers to common questions about the RFID linen tracking solution.
**Audience:** Hotel management, procurement decision-makers, non-technical stakeholders.
**Contents:** What the system does, how it works, what hardware is needed, how accuracy is achieved, and implementation considerations.

## Scope Note

These documents describe the locally verified demonstration scope. The platform has been verified through physical device testing (Chainway C5 E710) on a local network. Production deployment to `linen.alinktech.my.id` is intentionally deferred and is not claimed in these documents.
